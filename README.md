# Android-Memory
App

Nav:<br> Prev, goes to last item.
Next, goes to next item 
Start, goes to beginning .
Random, goes to random item.<br><br>

Actions:<br> 
Answer, displays answer.
Clear, clears screen.
Mem, flags item as memorized.
Skip, toggles memorized items as ignored.<br><br>

Tools:<br> Jump, searches for string in questions .
Fuzzy, sets percentage to find correct answers.
Stats, shows statistics.
Load File, opens data file.<br><br>
                Swipe:<br> Left / Right (Next / Prev)<br> Up (Rand)<br> Down (Start)<br> Tap (Answer)<br> Hold (Mem)
